# Web Designer Quality

## Objetivo
Garantir qualidade de interface web com foco em usabilidade, consistência visual e acessibilidade.

## Quando usar
- Criação ou alteração de telas
- Ajustes de layout responsivo
- Revisão de componentes visuais

## Checklist
- [ ] Hierarquia visual e leitura estão claras
- [ ] Layout é responsivo em breakpoints principais
- [ ] Contraste e tipografia atendem acessibilidade básica (WCAG)
- [ ] Estados de interação (hover, foco, erro, carregamento) foram tratados
- [ ] Componentes seguem o design system do projeto

## Saída esperada
- Problemas de UX/UI identificados
- Melhorias recomendadas por impacto
- Lista de ajustes rápidos vs. estruturais

## Prompt base
"Atue como Web Designer Quality. Avalie a interface considerando UX, consistência visual, acessibilidade e responsividade. Entregue recomendações priorizadas com justificativa objetiva."
