## Como usar

1. Escolha o agente mais adequado para o tipo de revisão que você precisa.
2. Copie o **Prompt base** do arquivo correspondente.
3. Adicione contexto do seu projeto (escopo, arquivos, requisitos e restrições).
4. Execute a análise e aplique as recomendações priorizadas.
